Class SerialRuleExecutor

An implementation of IExecutesAllRules which executed each rule in serial with no parallelisation.

Remarks

This service works by using a ITracksRuleDependencies instance for all of the available rules. This class uses GetRulesWhichMayBeExecuted() to get a collection of available rules (which either have no dependencies or where all dependencies have passed).

The logic in this class then executes all of these available rules and gets their results, storing them back against the original ExecutableRule object. Once all of the available rules have been executed to their logical completion, another iteration is performed using GetRulesWhichMayBeExecuted() to get the next collection of available rules.

This process continues until the dependency-tracker object fails to yield any available rules, at which point the rule execution stops and further results are added based upon those rules which have dependency failures.
