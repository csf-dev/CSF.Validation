Remarks

Any predicate parameters which are not specified or have a null value, are ignored. Any which are specified with a non-null value are used to filter the validation rule results. This means that Query() on its own with no parameters specified will return all of the validation rule results with no filtering.

QueryByMember<TValidated>(Expression<Func<TValidated, Object>>, Object, String, String, RuleOutcome)

Queries for validation rule results which validate a specified member of an object and which optionally also match all other specified predicates.

Type Parameters
Name Description
TValidated

The validated-object-type from which the member is available.
Remarks

This is used by specifying the validated-object-type which has the member (by which you wish to query) as the generic type parameter for this method, and then using an expression to select the member.

Any predicate parameters which are not specified or have a null value, apart from the memberAccessor which is mandatory in this method, are ignored. Any which are specified with a non-null value are used to filter the validation rule results.

Examples

If you wished to query for rules for the Age property of a validated Pet instance, then you would use this method like so. 

var results = queryableResult.QueryByMember<Pet>(x => x.Age);
