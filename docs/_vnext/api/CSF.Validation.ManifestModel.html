RelativeIdentifier

A model which may be used to indicate another rule, based upon a relative position within a validation hierarchy. This is used to indicate dependencies between validation rules. The models in this namespace provide a serialization-friendly mechanism by which to describe a validation manifest.

Rule

RuleConfigurationFactory

RuleTypeResolver
